Three.js browser driving prototype

Urban Drive

Drive, get out on foot, switch camera views, and run city missions in a stylized urban sandbox.

Vehicle controls
  • W accelerate / walk forward
  • S brake / reverse / walk back
  • A / D steer / strafe
  • Space handbrake
Interaction
  • E enter / exit car
  • C switch first / third person
  • Shift sprint on foot
  • H horn

Run from a local server such as VS Code Live Server for best results.
